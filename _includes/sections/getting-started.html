Getting Started

Using Jekyll and the Bootstrap framework and the pre-built components of {{ site.title }}, a web developer can easily put together a website mock-up or presentation. All the components that build a mockup with {{ site.title }} are laid out in a structural heirarchy.

Layout

A layout is a template or skeleton. Pages are assigned a layout and layouts can, and should be, reused. Examples of layouts are: a landing page without navigtion, single column with top navigation, two column with left navigation, etc.

Page

Pages have content specific to that page and are assigned a template. Pages could have endless configurations which may include copy or images using simple HTML, sections, or elements.

Section

Sections are areas of focus on a page or a thematic grouping of content. This page has several sections like the hero section at the top, the introduction section just below, and this one — the structure section.

Element

An element is a reusable item on a page. An element could be a button, a thumbnail image, or an HTML table.
